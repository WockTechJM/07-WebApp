# Expense Tracker

A simple web app to track your expenses by name, amount, and category.  
Built with HTML, CSS, and JavaScript.

## Features
- Add expenses with name, amount, and category
- Dynamic expense list updates
- Delete expenses
- Saves data in browser storage

## How to Use
1. Open `index.html` in your browser
2. Type the expense name and amount
3. Select a category from the dropdown
4. Click "Add Expense" to save it
5. Your expenses will show in the list below


## Technologies
- HTML
- CSS
- JavaScript
